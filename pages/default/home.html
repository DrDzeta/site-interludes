Les interludes, ou interENS ludiques, regroupent annuellement les étudiants des quatre ENS de France autour d'activités ludiques. Jeux de plateau, jeux de rôles, jeux vidéos, murders et autres sont à l'honneur durant ce week-end de trois jours.

Cette année, c'est au tour de l'{{ settings.hosting_school }} d'organiser les interludes. Elles auront lieu {% if settings.date_start %}{% if settings.date_end %} {% if settings.date_end.month != settings.date_start.month %} le week-end du {{ settings.date_start|date:"j F" }}-{{ settings.date_end|date:"j F Y" }}. {% else %} le week-end du {{ settings.date_start.day }}-{{ settings.date_end }}. {% endif %} {% else %} le week-end du {{ settings.date_start }}. {% endif %} {% else %} à une date non-fixée. {% endif %}
